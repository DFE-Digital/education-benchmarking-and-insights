# Front-end Components: Agent Mandates

This file defines AI-specific operational guidelines for agents working within the `front-end-components` module.
For all human-readable architecture, tech stack, development standards, and anti-patterns, refer to the `README.md` in this directory.

## AI Operational Guidelines

- **Types**: Use strict TypeScript typing for all props and state. Do not use `any`.
- **Testing**: Generate comprehensive tests using React Testing Library alongside any newly generated components.
