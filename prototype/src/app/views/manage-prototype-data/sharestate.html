Share the state of the prototype

How to use it

On any page click the    Share page    link in the footer.

What it does

It generates a link to reproduce the prototype in the same state.

Use case

In most tickets BAs wanted to share a specific state in the prototype with the Devs.

Using the query string worked but wasn't ideal because:

  • It doesn't save entered data that not saved to the session
  • it required fiddly coping and pasting and URL building

How it works

It's not tidy but it works

  1. Pressing the link in the footer trigger javascrip on the same page.
  2. The javascript then saves all the data on the page to the session
  3. That saved session data is then saved as an item in the session
  4. The page redicrects to a confirmtion page 'copy-url-to-clipboard.html'
  5. The confirmation page then automatically saves the outputted URL and state to the clipboard.

WARNING NOTE

This requires the user tell your browser once that this website can have 'Clipboard' permission.
