# AAR Specific notes

AAR has a transparency file which is released alongside the AAR data. 

## Gotchas

* Find all the ancillary data used in the AAR release in `get_aar_ancillary_data()`.
* SQL views for AAR submissions update midmorning on weekdays.
* A specific thing to check thoroughly in AAR is the system to apportion central service spend to academies. [There are a few docs on this at a business level.](../../data/09_Academy-Apportionments.md)
